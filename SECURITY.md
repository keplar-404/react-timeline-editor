# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| `1.0.x` (latest) | ✅ Yes |
| `< 1.0.0` | ❌ No |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public GitHub issue**.

Instead, report it privately by emailing the maintainer directly via the contact on the [GitHub profile](https://github.com/keplar-404).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You can expect an acknowledgement within **48 hours** and a resolution or status update within **7 days**.
